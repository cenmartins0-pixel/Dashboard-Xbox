# Dashboard-Xbox
Dashboard criado em planilha eletrônica para demonstrar as vendas de GAME PASS do Xbox.

Por meio de tabelas dinâmicas foi criado um dashboard que permite visualizar as vendas de GAME PASS do Xbox, do EA e Minecraft Season Pass. Também é possível filtrar os dados por tipo de assinatura.

Desta forma foi possível transformar dados em informação para possíveis tomadas de decisão.
